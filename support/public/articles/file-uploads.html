Files & Uploads

How file uploads work

Last updated May 2025

Todoer supports file uploads in both personal and workspace contexts. Access your files from the Files page via the sidebar navigation.

Uploading a file

  1. Navigate to the Files page from the sidebar
  2. Click Upload file and select a file using the standard file picker
  3. Click the Upload button to complete the upload

Uploaded files appear as cards showing the filename, file size, and upload date.

Note Each file is limited to 10 MB. If you need to share something larger, consider using an external file sharing service and pasting the link into a task.

File types

Todoer accepts files of all types — there are no restrictions on file extensions. Whatever you need to share, you can upload it.

Working with archives

Todoer supports .zip, .tar, and .gz archives. After uploading an archive, click the Extract button on its card to extract its contents manually. Extraction is not automatic.

You can also create archives from selected files directly within Todoer.

Downloading and deleting files

Click the download icon on any file card to download it. To delete a file, use the delete option on the card.

In a workspace, owners and admins can delete any file. Members can delete files they uploaded. Viewers cannot delete files.

Deleted files are permanently removed and cannot be recovered.

Image previews

Image files can be previewed directly in Todoer. Click an image file to open a preview modal with zoom controls.

File visibility

Files are visible to all members of the workspace. There is currently no per-file permission control — if a user has access to the workspace, they can see and download all files in it.
